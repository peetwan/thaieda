### What to add — TEXT-SPECIFIC INSIGHTS:

Add new functions that generate insights from text metrics and NER results:

1. **`_text_length_distribution_insights`**: From text_metrics, check if text length distribution is unusual:
   - If avg char length varies significantly across rows (high std) → insight about variability
   - If there are very short or very long texts (outliers in length) → insight
   - If median length is very short (<10 chars) → "ข้อมูลเป็นข้อความสั้น ๆ (เช่น social media)"
   - If median length is very long (>200 chars) → "ข้อมูลเป็นรีวิว/บทความยาว"

2. **`_vocabulary_richness_insight`**: From text_metrics:
   - If unique_tokens / total_tokens ratio is low (<0.1) → "คำซ้ำเยอะ คำศัพท์หลากหลายต่ำ"
   - If ratio is high (>0.5) → "คำศัพท์หลากหลายสูง"
   - This uses `text_metrics.unique_tokens` and `text_metrics.total_tokens`

3. **`_ner_summary_insight`**: From NER results (if available):
   - If total_entities > 0 → "พบ named entities N ตัว บ่งชี้ข้อมูลมีความหลากหลายของเนื้อหา"
   - If top entity type is PERSON → "ข้อมูลมีชื่อบุคคลเป็น entities หลัก"
   - If top entity type is LOCATION → "ข้อมูลเกี่ยวข้องกับสถานที่"

4. **`_sentiment_distribution_insight`**: If there's a rating/sentiment column:
   - Check distribution skew of ratings
   - If mostly 5-star → "ส่วนใหญ่เป็นรีวิวเชิงบวก"
   - If mostly 1-star → "ส่วนใหญ่เป็นรีวิวเชิงลบ — อาจเป็นปัญหาที่ต้องตรวจสอบ"
   - If bimodal (lots of 1-star and 5-star) → "คะแนนแบ่งเป็น 2 กลุ่มชัดเจน"

5. **`_language_mixing_insight`**: From quality checks or text metrics:
   - If text has significant Latin script mixing → "พบการผสมภาษาไทย-อังกฤษในข้อมูล"
